# AFCAT Readiness Tracker

## Project Overview

AFCAT Readiness Tracker is a web-based application developed using HTML, CSS, and JavaScript. The application helps AFCAT aspirants evaluate their preparation status through eligibility checking, study progress tracking, fitness assessment, and readiness analysis.
The project demonstrates the implementation of modern web development concepts including Flexbox, Grid Layout, HTML5 APIs, JavaScript Event Handling, Forms, Validation, and Responsive Design.

---

## Problem Statement

Students preparing for the Air Force Common Admission Test (AFCAT) often lack a simple platform to track their eligibility, study progress, and fitness readiness.
This project provides an interactive solution that allows users to assess their preparation level and monitor their readiness for AFCAT.

---

## Features

### 1. Eligibility Checker

* Accepts age, height, and degree information
* Validates user eligibility based on predefined criteria
* Displays eligibility status dynamically

### 2. Study Progress Tracker

* Allows users to enter scores for:

  * Mathematics
  * English
  * Reasoning
  * General Knowledge
* Calculates overall readiness score

### 3. Fitness Tracker

* Tracks:

  * Pushups
  * Running Distance
  * BMI
* Evaluates physical fitness status

### 4. Geolocation API

* Uses the HTML5 Geolocation API
* Retrieves and displays the user's current location coordinates

### 5. AFCAT Readiness Assessment

* Implemented using a Modal Popup
* Calculates readiness percentage using:

  * Mock Test Score
  * Study Hours
  * Fitness Score

### 6. Responsive Design

* Adapts to different screen sizes
* Uses media queries for mobile compatibility
